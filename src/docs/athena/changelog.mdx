Athena changelog. Check the latest updates and see what's new.

**Version**: 1.3.0

**Date of release (V1.3.0)**: 01/07/20 GTM – 6

## Version 1.3.0

- **Added**: Gulp Integration.

## Version 1.2.0

- **Added**: Support for Ghost Membership / Subscription Model.
- **Added**: Full Compatibility with Ghost 3. Please update to the latest Ghost ## Version.
- **Added**: Support for Bookmark Cards.
- **Enhanced**: The theme is now more easy to edit and require less steps to setup some features.
- **Fixed**: Translation issues.
- **Fixed**: Minor Issues.
- **Removed**: HTML Documentation. The previous docs were changed by a PDF file.
- **Removed**: The additional Athena version with color switcher. You can now easily hide the color switcher.

## Version 1.1.6

- **Eliminated**: Support for Ghosts Blogs with the Old API. Please update to the latest Ghost ## Version.
- **Fixed**: Changed {{lang}} to {{@site.lang}} as it is required by Ghost v. 2.19.4.
- **Fixed**: GScan’s errors and warnings.
- **Fixed**: Post tags spacing.

## Version 1.1.5

- **Added**: Styles for tables.
- **Tweaked**: Eliminated scrollbars on carousel and vertical navigation bar.
- **Fixed**: Missing ‘get’ parameters.
- **Fixed**: Spacing on tags.
- **Fixed**: Masonry grid layout spacing issue.
- **Fixed**: Image Galleries not preserving image proportions.

## Version 1.1.4

- **Added**: French translation (Thanks to ASX31).
- **Fixed**: Infinite Scroll was only displaying one post on each request.

## Version 1.1.3

- **Fixed**: Search Functionality was broken due to new Content API. Now, it requires users to generate an API key in order to use it (Check documentation). This update is necessary only for people using Ghost ## Version 2.10.2 or newer and the new Ghost’s API.

## Version 1.1.2

- **Added**: Support for Content API v2.
- **Updated**: Installation instructions on documentation files.

## Version 1.1.1

- **Added**: Instagram Widget.
- **Fixed**: Minor bugs.

## Version 1.1.0

- **Added**: Horizontal Navbar.
- **Added**: Dark Color Scheme.
- **Added**: New background pattern for Dark ## Version.
- **Tweaked**: Collapsables now activate by click and not by a hover event.
- **Fixed**: Minor bugs.
- **Fixed**: Tags Spacing Issue.
- **Fixed**: GScan warning. Theme is now 100% valid.

## Version 1.0.1

- **Fixed**: Error page not loading assets and scripts.
- **Fixed**: Color and Alignment Issues.

## Version 1.0.0

- Release.
